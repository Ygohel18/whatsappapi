## WhatsApp API using venom-bot

## Install
```npm i```

## Setup
Create .env file

```PORT={{port}}```

```TOKEN={{Your API Token}}```

```WEBHOOK={{webhook1}},{{webhook2}}```


## Deploy
```Local: npm run local```

```Production: npm run start```
